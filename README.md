# numer0n-search
numer0n search project
numer0n戦略全探索プロジェクト
最強質問戦略を作るためにc言語を用いて質問候補の探索を行う。

### コマンド一覧
Linux環境推奨
* `make all` ビルド（`make`可）
* `make out` プログラムの実行
* `make clean` 一時ファイルと実行ファイルの削除
